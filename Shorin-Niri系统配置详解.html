2. 主题联动机制

2.1 Matugen 颜色提取器

系统使用 matugen 从当前壁纸提取颜色，通过模板系统自动生成各个组件的配色文件，实现全局主题联动。

配置文件：~/.config/matugen/config.toml

支持的配色方案：

  • scheme-tonal-spot（默认）
  • scheme-content / scheme-expressive / scheme-fidelity
  • scheme-fruit-salad / scheme-monochrome / scheme-neutral
  • scheme-rainbow / scheme-vibrant

切换配色方案快捷键：Mod+Alt+T

2.2 Waypaper 壁纸管理

配置文件：~/.config/waypaper/config.ini

[Settings]
folder = ~/图片/Wallpapers
backend = swww
post_command = $HOME/.config/scripts/matugen-update.sh $wallpaper & sleep 0.8 & $HOME/.config/scripts/niri_set_overview_blur_dark_bg.sh

post_command 换壁纸后自动执行：

  1. 调用 matugen-update.sh 提取颜色并更新主题
  2. 调用 niri_set_overview_blur_dark_bg.sh 生成 overview 模糊背景

为什么 GTK 应用和 Yazi 颜色也跟着变？
GTK 应用（如 Nautilus）：matugen 生成 ~/.config/gtk-3.0/colors.css 和 ~/.config/gtk-4.0/colors.css，GTK 主题通过 @import url("colors.css") 引入这些颜色变量。当前 GTK 主题：adw-gtk3-dark
Yazi：matugen 直接生成完整的主题文件 ~/.config/yazi/theme.toml，yazi 启动时自动加载。

3. 终端配置

3.1 Fish Shell

配置文件：~/.config/fish/config.fish

f 命令详解

在终端输入 f 会自动执行 fastfetch 并显示一张固定图片。

定义位置：~/.config/fish/functions/f.fish

function f
    set -l FIXED_IMG "/home/LinXuan/图片/IMG_20260405_205718.jpg"
    if test -f "$FIXED_IMG"
        fastfetch --logo "$FIXED_IMG" --logo-preserve-aspect-ratio true $argv
    else
        echo "图片找不到了 QAQ"
        fastfetch $argv
    end
end

工作原理：

  1. Fish 会在启动时自动加载 ~/.config/fish/functions/ 下的所有 .fish 文件
  2. 每个文件定义一个与文件名同名的函数
  3. config.fish 最后一行有 f，所以终端启动时自动执行

其他自定义命令

命令功能
yyazi 文件管理器（退出后自动跳转到浏览目录）
lseza（带图标的 ls 替代）
sl蒸汽火车动画（带彩虹效果）
fa快速运行 fastfetch
raw随机下载动漫壁纸

5. 截图与录屏功能

截图功能

使用工具：niri 内置截图 + grim + slurp + satty

快捷键功能
Print 或 Mod+Alt+A选取区域截图
Shift+Print截图整个显示器
Ctrl+Print截图当前窗口
Mod+E截图后用 satty 编辑

截图保存位置：~/图片/Screenshots/%Y-%m-%d %H-%M-%S.png

截图音效：由 ~/.config/niri/scripts/screenshot-sound.sh 守护进程提供。

录屏功能

使用工具：wf-recorder

配置方式：通过顶部任务栏的录屏模块操作（无独立配置文件，通过命令行参数控制）。

# 录制整个屏幕
wf-recorder -f ~/视频/recording.mp4

# 录制选定区域
wf-recorder -g "$(slurp)" -f ~/视频/recording.mp4

# 录制带音频
wf-recorder -audio -f ~/视频/recording.mp4

6. 输入法配置

配置目录：~/.config/fcitx5/

主要文件：

  • config — 主配置（快捷键等）
  • profile — 输入法列表
  • conf/ — 各模块配置

输入方案：

  • 键盘 — 英语
  • Rime（雾凇拼音 rime-ice）

切换快捷键

操作快捷键
切换输入法Super+空格
激活/取消激活输入法双击左 Ctrl
重启输入法（解决卡死）Mod+F1

环境变量（在 niri 的 config.kdl 中设置）

GTK_IM_MODULE "fcitx"
QT_IM_MODULE "fcitx"
XMODIFIERS "@im=fcitx"

主题：由 matugen 自动生成到 ~/.local/share/fcitx5/themes/Matugen/

启动方式：spawn-at-startup "fcitx5" "-d"

常见问题：如果输入法无法使用，按 Mod+F1 重启 fcitx5。如果出现"卡 A"的情况，试试按右 Shift 解决。

7. 剪贴板配置

剪贴板工具：shorinclip（TUI 界面）

程序位置：/usr/bin/shorinclip

缓存目录：~/.cache/shorinclip/

特点：

  • TUI 界面（终端用户界面）
  • 支持图片预览（kitty 终端）
  • 支持搜索和过滤
  • 颜色自动适配 matugen 主题

工作流程

  1. wl-paste -watch cliphist store 监听剪贴板变化
  2. 每次复制都会保存到 cliphist 历史
  3. shorinclip 读取 cliphist 历史并显示
快捷键功能
Mod+Alt+V打开剪贴板历史

操作方式：打开后显示历史记录列表 → 使用方向键或搜索选择项目 → 回车确认选择，内容自动粘贴。

9. 护眼模式配置

工具：wlsunset

控制脚本：~/.local/bin/toggle-wlsunset

配置参数：

TEMP_LOW="5000"      # 夜间/护眼色温（开尔文）
TEMP_HIGH="6500"     # 日间/标准色温
TRANSITION="1800"    # 过渡时间（30 分钟）
LAT="36.5"           # 纬度
LONG="128.0"         # 经度

工作原理：

  • 根据经纬度自动计算日出日落时间
  • 日落后逐渐降低色温（偏黄，护眼）
  • 日出后恢复标准色温
快捷键功能
Mod+F9护眼模式开关

12. 常见问题与排错

双任务栏问题

症状：屏幕出现两个任务栏
解决：systemctl --user disable --now dms

网络问题

症状：WiFi 连接异常
解决（更换 WiFi 后端为 wpa_supplicant）： 
sudo rm /etc/NetworkManager/conf.d/iwd.conf
sudo systemctl restart NetworkManager

输入法问题

症状：输入法无法使用
解决 1：按 Mod+F1 重启 fcitx5
解决 2：出现"卡 A"时按右 Shift 解决

回档方法

如果不想要了或者安装失败了：
/usr/local/bin 下有两个脚本可以回档到运行脚本之前的状态：
  • 回到安装桌面前：shorin-de-undochange
  • 回到运行脚本前：shorin-undochange

Overview 壁纸不显示

步骤：
  1. Mod+Z 打开程序菜单，打开 DMS 设置
  2. 进入"个性化 → 壁纸"，设置一张壁纸
  3. 开启"带模糊效果的壁纸复本"
  4. Mod+O 打开 overview 即可看到模糊壁纸效果

GTK 主题设置

步骤：
  1. 选择"主题与配色"，点击 auto，选择喜欢的配色方案
  2. 在页面底端点击"应用 GTK 配色"
  3. 打开 GTK Setting 软件，选择使用 adw-gtk3-dark

自动熄屏锁屏

swayidle 脚本默认被注释掉了。如需启用自动熄屏锁屏睡眠：
  1. 编辑 ~/.config/niri/config.kdl
  2. 找到 spawn-at-startup "~/.config/niri/scripts/swayidle.sh"
  3. 取消注释，重启 niri

命令功能
