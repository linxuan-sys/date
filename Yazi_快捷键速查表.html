Yazi 文件管理器 - 快捷键速查表

适用于轩轩的配置文件

一、基本操作
q退出程序 Q退出且不输出 cwd
Esc退出模式/取消 Ctrl+c关闭标签页
Ctrl+z挂起程序 ~ / F1打开帮助
二、光标移动
k / ↑上一个文件 j / ↓下一个文件
Ctrl+u上翻半页 Ctrl+d下翻半页
Ctrl+b上翻一页 Ctrl+f下翻一页
g g跳到顶部 g t跳到底部
三、目录导航
h / ←返回上级目录 l / →进入子目录
Shift+←历史后退 Shift+→历史前进
四、目录跳转（先按 g）
h主目录 ~/ d下载目录
p图片目录 v视频目录
m音乐目录 w文档目录
u公共目录 t模板目录
Space交互式输入路径 f跟随符号链接
五、文件操作
Enter打开文件 o o打开文件
O交互式打开（选择程序） o c用自定义命令打开
y复制文件 x剪切文件
p p粘贴 p f强制粘贴（覆盖）
-创建符号链接（绝对路径） _创建符号链接（相对路径）
Y / X取消复制/剪切 d移到回收站
D永久删除 ⚠️ a创建文件/目录
;后台运行命令 :阻塞运行命令
六、文件重命名（先按 r）
r光标在扩展名前 f清空整个文件名
i光标在开头 a光标在末尾
c清空整个文件名
七、复制路径（先按 c）
c复制完整路径 d复制目录路径
f复制文件名（含扩展名） n复制文件名（不含扩展名）
m重命名（保留扩展名） r创建文件/目录
z压缩文件
八、选择操作
Space切换选择 Ctrl+a全选
Ctrl+r反选 v视觉模式（选择）
V视觉模式（取消选择）
九、搜索查找
f智能过滤 /查找下一个
?查找上一个 n下一个匹配
N上一个匹配 sfd 搜索文件名
Sripgrep 搜索内容 Ctrl+s取消搜索
zfzf 模糊跳转 Zzoxide 智能跳转
十、显示设置（先按 m）
.切换隐藏文件显示
s显示文件大小 p显示权限
b显示创建时间 m显示修改时间
o显示所有者 n无额外信息
十一、排序（先按 ,）
m/M修改时间（正/倒序） b/B创建时间（正/倒序）
e/E扩展名（正/倒序） a/A字母顺序（正/倒序）
n/N自然排序（正/倒序） s/S文件大小（正/倒序）
r随机排序
十二、标签页管理
t新建标签页 1-9切换到第N个标签页
-上一个标签页 =下一个标签页
_与上一个交换 +与下一个交换
十三、预览控制
Ctrl+↑预览向上滚动 Ctrl+↓预览向下滚动
Tab聚焦模式（查看详情）
十四、回收站
R b打开回收站管理
十五、挂载设备（先按 M）
m选择设备挂载并跳转 R重新挂载当前设备
u/U卸载并弹出（U强制） a添加挂载 URI
e编辑挂载 URI r移除挂载 URI
p通用挂载插件
G m跳转到设备挂载点（自动） ` `跳回上一个目录
十六、书签管理 Whoosh（先按 ]）
[按键跳转书签 }模糊搜索书签
a/A添加书签（A仅当前目录） t/T添加临时书签
f模糊搜索跳转 d按键删除书签
D模糊搜索删除（可多选） C删除所有书签 ⚠️
r/R重命名书签 Alt+k直接跳转到书签 'k'
十七、任务管理器（按 w 打开）
Esc / w关闭 j / k上/下一个任务
Enter查看详情 x取消任务
十八、聚焦模式（按 Tab 进入）
Esc / Tab退出 j / k上/下一行
h / l上/下一个文件 c c复制单元格
十九、选择器模式
Esc取消 Enter确认
j / k上/下一个选项
二十、输入模式
Enter提交输入 Esc / Ctrl+c取消
h/l/←/→左/右移一字符 b / w上/下一个单词
0 / Home移到行首 $ / End移到行尾
Backspace删除前一字符 Delete删除当前字符
Ctrl+u删除到行首 Ctrl+k删除到行尾
Ctrl+w删除前一单词 Ctrl+d删除当前字符
u撤销 Ctrl+r重做
d剪切选中 D剪切到行尾
y复制选中 p粘贴到光标后
P粘贴到光标前
二十一、确认对话框
y / Enter确认 n / Esc取消
二十二、补全模式
Tab选择补全 Enter选择并确认
↑/↓/Alt+j/k切换补全项 Ctrl+c取消
常用速记
j/k上下移动 h/l进出目录
y/x/p p复制/剪切/粘贴 d删除到回收站
