文档格式转换命令速查表

使用 pandoc 进行格式转换，OnlyOffice 用于 PDF 相关操作

一、Markdown 转换

MD → Word

pandoc 文件.md -o 文件.docx

MD → HTML

pandoc 文件.md -o 文件.html

MD → PDF

# 方法一：先转 HTML，再在浏览器中打开 HTML，按 Ctrl+P 打印为 PDF
pandoc 文件.md -o 文件.html

# 方法二：先转 Word，再用 OnlyOffice 导出 PDF
pandoc 文件.md -o 文件.docx
# 然后用 OnlyOffice 打开 Word，文件 → 另存为 → PDF

MD → 纯文本

pandoc 文件.md -o 文件.txt

MD → RTF

pandoc 文件.md -o 文件.rtf

MD → EPUB 电子书

pandoc 文件.md -o 文件.epub

MD → ODT（LibreOffice 格式）

pandoc 文件.md -o 文件.odt

二、Word 转换

Word → MD

pandoc 文件.docx -o 文件.md

Word → HTML

pandoc 文件.docx -o 文件.html

Word → PDF

使用 OnlyOffice：
1. 打开 OnlyOffice
2. 文件 → 打开 → 选择 Word 文件
3. 文件 → 另存为 → 选择 PDF 格式
4. 保存

Word → EPUB

pandoc 文件.docx -o 文件.epub

Word → ODT

pandoc 文件.docx -o 文件.odt

三、HTML 转换

HTML → MD

pandoc 文件.html -o 文件.md

HTML → Word

pandoc 文件.html -o 文件.docx

HTML → PDF

在浏览器中打开 HTML 文件：
1. 用 Chrome 或 Firefox 打开 HTML 文件
2. 按 Ctrl+P 打印
3. 目标打印机选择"另存为 PDF"
4. 保存

HTML → EPUB

pandoc 文件.html -o 文件.epub

四、PDF 转换

PDF → Word

使用 OnlyOffice：
1. 打开 OnlyOffice
2. 文件 → 打开 → 选择 PDF 文件
3. OnlyOffice 会自动将 PDF 转换为可编辑文档
4. 文件 → 另存为 → 选择 DOCX 格式
5. 保存

PDF → MD

pandoc 不支持直接转换，需分两步：
1. 用 OnlyOffice 将 PDF 转为 Word
2. 再用 pandoc 将 Word 转为 MD
   pandoc 文件.docx -o 文件.md

PDF → HTML

pandoc 不支持直接转换，需分两步：
1. 用 OnlyOffice 将 PDF 转为 Word
2. 再用 pandoc 将 Word 转为 HTML
   pandoc 文件.docx -o 文件.html

五、其他格式

EPUB 转换

# EPUB → MD
pandoc 文件.epub -o 文件.md

# EPUB → Word
pandoc 文件.epub -o 文件.docx

# EPUB → HTML
pandoc 文件.epub -o 文件.html

ODT 转换

# ODT → MD
pandoc 文件.odt -o 文件.md

# ODT → Word
pandoc 文件.odt -o 文件.docx

# ODT → HTML
pandoc 文件.odt -o 文件.html

六、常用参数

添加页边距

pandoc 文件.md -o 文件.docx -V geometry:margin=1in

指定编码

pandoc 文件.md -o 文件.html --metadata title="标题"

从 URL 转换

pandoc https://example.com -o 文件.html

合并多个文件

pandoc 文件1.md 文件2.md -o 合并.docx

七、查看支持的格式

# 查看支持的输入格式
pandoc --list-input-formats

# 查看支持的输出格式
pandoc --list-output-formats

八、快捷方式总结

源格式 目标格式 方法
MD Word pandoc 文件.md -o 文件.docx
MD HTML pandoc 文件.md -o 文件.html
MD PDF 先转 HTML/Word，再用浏览器/OnlyOffice
Word MD pandoc 文件.docx -o 文件.md
Word HTML pandoc 文件.docx -o 文件.html
Word PDF OnlyOffice 打开，另存为 PDF
HTML MD pandoc 文件.html -o 文件.md
HTML Word pandoc 文件.html -o 文件.docx
HTML PDF 浏览器打开，Ctrl+P 打印为 PDF
PDF Word OnlyOffice 打开，另存为 DOCX
PDF MD OnlyOffice 转 Word，再 pandoc 转 MD
